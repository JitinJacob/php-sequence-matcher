# php-sequence-matcher

A longest sequence matcher. The logic is primarily based on the Python [difflib](https://docs.python.org/3/library/difflib.html) package.


# Requirements

- PHP >= 7.1.3


# Installation

```
$ composer require jfcherng/php-sequence-matcher
```
